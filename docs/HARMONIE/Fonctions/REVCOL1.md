# REVCOL

## Fonction REVCOL

La fonction **REVCOL** inverse l'ordre des colonnes d'une variable : Le 1er élément devient le dernier, le 2ème l'avant-dernier, etc.

#### Syntaxe :&nbsp;

Q01.REVCOL()

ou

\_REVCOL(Q01)
