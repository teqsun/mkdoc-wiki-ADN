# Highcol

## Highcol Function

The setpoint **Highcol** Makes it possible to highlight columns of a variable.

The arguments corresponds to the lists of elements to be bold.each list follows the syntax relating to items selections.

### Syntax: & nbsp;

Q01.highcol (Keys)
