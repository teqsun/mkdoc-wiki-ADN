# Day

## Day Function

The **Day** Function Builds A Digital Variable Returning the "Day" Component of the date placed in argument.

### Syntax: & nbsp;

Q01.Day ()

Gold

\ _Day (Q01)

### EXAMPLES:

Dates.lay ()

Decodes The Temporal Variable Dates and Returns The Value of the Days.
