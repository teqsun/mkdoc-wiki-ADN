# Titleize

## Titleize Function

The **Titleize** function passes all the characters from a literal variable to lower case with the exception of the first character which passes in capital letters.

### Syntax: & nbsp;

Q01.titleize ()

Or

\ _Titleize (Q01)

& nbsp;

Applied to a variable literal.

& nbsp;
