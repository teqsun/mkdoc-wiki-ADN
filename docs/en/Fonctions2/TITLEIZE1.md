# Titleize

## Titleize Function

The **Titleize** Function Passes All The Characters from A Literal Variable to Lower Case With the Exception of The First Character Which Pass in Capital Letters.

### Syntax: & nbsp;

Q01.titleize ()

Gold

\ _Titleize (Q01)

& nbsp;

Applied to a variable literal.

& nbsp;
