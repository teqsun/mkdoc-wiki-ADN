# UPPER

## Fonction UPPER

La fonction **UPPER** passe tous les caractères des réponses à une variable littérale en majuscules.

#### Syntaxe :&nbsp;

Q01.UPPER()

ou

\_UPPER(Q01)

&nbsp;

S'applique à une variable littérale
