# REVROW

## Fonction REVROW

La fonction **REVROW** inverse les lignes de la variable traitée : Le 1er élément devient le dernier, le 2ème l'avant-dernier, etc.

#### Syntaxe :&nbsp;

Q01.REVROW()

ou

\_REVROW(Q01)
